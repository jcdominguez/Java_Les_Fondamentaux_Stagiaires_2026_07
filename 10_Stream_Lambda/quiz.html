Le scénario fil rouge

Vous rejoignez l'équipe qui développe un outil d'analyse de la base clients d'une entreprise. Section A : on vérifie les concepts. Section B : on écrit les briques du projet. Section C : on relit les pipelines assemblés par l'équipe. Section D : l'outil passe aux statistiques (agrégations & Collectors — avancé).

Toutes les questions s'appuient sur la classe Client et les données suivantes (un client est premium si ses dépenses dépassent 1000 EUR) :

public class Client {
    private String nom;      // getNom()
    private String ville;    // getVille()
    private int age;         // getAge()
    private double depenses; // getDepenses()

    @Override
    public String toString() {
        return nom + " (" + ville + ", " + age + " ans, " + depenses + " EUR)";
    }
}

List<Client> clients = List.of(
    new Client("Alice", "Paris", 32, 1500.0),
    new Client("Bob", "Lyon", 25, 800.0),
    new Client("Charlie", "Paris", 45, 2500.0),
    new Client("Diana", "Marseille", 28, 1200.0),
    new Client("Eve", "Lyon", 35, 950.0),
    new Client("Marc", "Paris", 52, 3200.0)
);
